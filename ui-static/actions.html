ACTIONS

These tools are for the advanced use.
Please refer to the documentation for more information: https://docs.overvis.com/#/OPCB/User_Docs/Guides/Web_Interface

Restart the OPCB runtime

Restart OPCB runtime, reloading the configuration files and all services. The connection to the device will be lost temporarily after this action.

Reboot the device

Soft-reboot of the device operating system. The connection to the device will be lost for several minutes after this action.

Reset to the default configuration

OPCB runtime configuration will be restored to the factory values after this action. The connection to the device may be lost after this action. OPCB Wi-Fi will automatically switched to Access Point mode.
