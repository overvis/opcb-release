MANUFACTURING

Restore identity with the private key

Warning! This will not change the local operations, but the device will acquire other device's identity on remote servers. This operation will communicate with remote server to register the identity of the device. Only do this if you have lost the device with specified private key or have restored its firmware to the factory firmware. Two devices with the same identity will conflict and will not be able to communicate with remote servers.

Register device

This form is for registering a new device on the remote Overvis servers. It requires manufacturer license to work. If you don't have a manufacturer license but want to register the device, please contact us: support@overvis.com.

Run tests

To run the tests please ensure the following:
  • Device has an access to the internet and is connected to Overvis VPN.
  • All device settings have default values.
  • On RS-485-1 port there is a Novatek-Electro OB-215 connected with Modbus address 1.
